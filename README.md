## Sobre o Cl Money

O Cl Money é uma alternativa simples e eficaz para gerenciar suas finanças, permitindo o cadastro de entradas, saídas e a organização das mesmas em categorias.

## Instalação e uso

```bash
# Abra um terminal e copie este repositório com o comando
git clone https://github.com/martins-rafael/dtmoney.git
# ou use a opção de download.
# Entre na pasta web com 
cd dtmoney
# Instale as dependências
yarn install
# Rode a aplicação
yarn start
```
